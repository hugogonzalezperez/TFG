1. Abre VS Code: src/index.css

2. Presiona: Ctrl+F

3. Busca: :root {

4. Encontrarás (línea ~2390):
   :root {
     --primary: #ac1279;         ← MORADO (color principal)
     --secondary: #10b981;       ← VERDE (secundario)
     --accent: #f59e0b;          ← NARANJA (destacado)
     --background: #ffffff;      ← BLANCO (fondo)
     --foreground: #1a1a1a;      ← NEGRO (texto)
     ...más variables
   }

5. Cambia el que quieras:
   ANTES:  --primary: #ac1279;
   DESPUÉS: --primary: #FF0000;   (Rojo)

6. Guarda: Ctrl+S

7. Recarga navegador: F5

8. ✨ ¡Cambio inmediato!
